# To-Do-List
**About**: This project is a simple, yet efficient, To-Do list web application designed to help users organize their daily tasks and boost productivity. It provides a seamless way to add, manage, and track tasks.
**Features**:
Add your school, work, daily tasks
Mark tasks as complete or incomplete.
Delete tasks from the list.
Filter tasks by their deadline that you can assign yourself
Save tasks locally in the browser's storage, so they persist across sessions.
User-friendly and responsive interface with colors that reflect the priority of the task that shows red for daily tasks, yellow for this week's tasks, and green for tasks more than a week and less than a month.
**Technologies Used:**
HTML5
CSS
JavaScript
